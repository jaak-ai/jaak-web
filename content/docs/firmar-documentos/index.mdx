# Firmar documentos

JAAK Signa es nuestra API de firma electrónica que permite firmar documentos con validez legal en México. Soporta múltiples tipos de firma según el nivel de seguridad y cumplimiento que requiera tu caso de uso.

## Tipos de firma disponibles

| Tipo | Verificación | Validez legal | Caso de uso |
|------|--------------|---------------|-------------|
| **Simple** | Ninguna | Baja | Términos y condiciones, avisos de privacidad |
| **Avanzada (OTP)** | Email o SMS | Media | Contratos de bajo riesgo, acuerdos internos |
| **Biométrica** | KYC facial con prueba de vida | Alta | Contratos financieros, arrendamientos |
| **FIEL** | Certificado SAT | Muy alta | Facturas, documentos fiscales |
| **PSC** | Prestador de servicios de certificación | Muy alta | Documentos legales críticos |

<Callout type="tip" title="Recomendación">
Para la mayoría de casos de uso comerciales, la firma **biométrica** ofrece el mejor balance entre seguridad, validez legal y experiencia de usuario.
</Callout>

## Flujo básico de firma

El proceso de firma electrónica sigue estos pasos:

```
1. Crear plantilla → 2. Crear envío → 3. Firmar → 4. Descargar
```

1. **Crear plantilla**: Sube un documento PDF y define las zonas de firma
2. **Crear envío (submission)**: Genera un enlace de firma con los datos del firmante
3. **Firmar**: El firmante completa la verificación de identidad y firma
4. **Descargar**: Obtén el documento firmado con el certificado de firma
