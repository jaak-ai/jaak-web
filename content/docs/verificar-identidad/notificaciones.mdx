La plataforma JAAK te permite configurar **notificaciones automáticas** para mantenerte informado sobre el estado de cada sesión de verificación KYC que se realice. Esta función te ayuda a reaccionar de forma rápida ante verificaciones **aprobadas**, **rechazadas** o **en revisión**, sin necesidad de consultar constantemente el panel de sesiones.


> **Ruta:** Plataforma JAAK → KYC → Notificaciones


![KYC Notificaciones](https://files.readme.io/87940b40cbbd6a07e3fbc0f912dc4bc6b70ff73f0770db54014037035e93fe85-KYC-Notificaciones.png)


## 1. Activar Notificaciones de Resultado

Puedes seleccionar qué tipo de resultados deseas recibir por correo electrónico. Cada vez que un usuario complete una sesión KYC, se enviará una notificación si se activa alguna de las siguientes opciones:

| Resultado         | Descripción                                                                                                                  |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| **Aprobado**      | La sesión fue validada correctamente.                                                                                        |
| **Rechazado**     | La sesión no cumplió los requisitos necesarios y fue rechazada.                                                              |
| **En Revisión**   | El sistema detectó posibles inconsistencias por lo que es necesario un proceso de análisis y requiere una aprobación manual. |

> Marca las casillas correspondientes para **activar** o **desactivar** cada tipo de notificación según tus necesidades.

## 2. Configurar Correos Destinatarios

Puedes agregar hasta **3 direcciones de correo electrónico** que recibirán las notificaciones seleccionadas en la sección anterior. Esto es útil si quieres mantener informados a otros miembros de tu equipo (analistas, supervisores, o responsables de compliance).

Haz clic en **"Agregar correo electrónico"**.

1. Ingresa la dirección deseada.
2. Repite el proceso hasta un máximo de 3 correos.
3. Guarda los cambios.

> Todos los correos registrados recibirán notificaciones únicamente de los **tipos de resultado** activados en tu configuración.


## Recomendaciones

* Asegúrate de que los correos ingresados estén activos y monitoreados.
* Puedes actualizar esta lista en cualquier momento para añadir o eliminar destinatarios.
* Combina esta funcionalidad con los **Webhooks KYC** para una integración más robusta.
