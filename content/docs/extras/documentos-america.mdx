## Notas Importantes

**OCR:** Reconocimiento Óptico de Caracteres - Extracción automática de datos del documento

**Verificación:** Validación de elementos de seguridad y autenticidad del documento

**Limitaciones Especiales:**
- **Solo VIZ:** Únicamente zona visual del documento
- **Solo Anverso:** Procesamiento del frente del documento únicamente
- **Beta:** Funcionalidad en desarrollo, puede presentar limitaciones
- **AAMVA:** Estándar estadounidense que permite procesamiento automático de documentos compatibles
