---
title: Lista del tipo de documentos
description: Catálogo de valores que representan la clasificación del documento procesado por los servicios de JAAK.
category: recurso
order: 4
---

El tipo de documento es un valor que representa la clasificación del documento procesado. A continuación, se listan los valores posibles junto con su significado.

| Tipo de Documento                  | Descripción                                                                   |
| ---------------------------------- | ----------------------------------------------------------------------------- |
| `TYPE_NONE`                        | Tipo de documento no identificado o no reconocido.                            |
| `TYPE_CONSULAR_ID`                 | Identificación emitida por una oficina consular.                              |
| `TYPE_DL`                          | Licencia de conducir.                                                         |
| `TYPE_DL_PUBLIC_SERVICES_CARD`     | Tarjeta de servicios públicos que también funciona como licencia de conducir. |
| `TYPE_EMPLOYMENT_PASS`             | Permiso de trabajo emitido a empleados extranjeros.                           |
| `TYPE_FIN_CARD`                    | Tarjeta de Identificación Financiera (Singapur).                              |
| `TYPE_ID`                          | Documento de identidad nacional.                                              |
| `TYPE_MULTIPURPOSE_ID`             | Documento de identidad para múltiples propósitos.                             |
| `TYPE_MyKad`                       | Tarjeta de identidad nacional de Malasia.                                     |
| `TYPE_MyKid`                       | Tarjeta de identidad para niños en Malasia.                                   |
| `TYPE_MyPR`                        | Tarjeta de residencia permanente en Malasia.                                  |
| `TYPE_MyTentera`                   | Tarjeta de identificación militar en Malasia.                                 |
| `TYPE_PAN_CARD`                    | Documento fiscal en India (Permanent Account Number).                         |
| `TYPE_PROFESSIONAL_ID`             | Identificación profesional para ciertas ocupaciones.                          |
| `TYPE_PUBLIC_SERVICES_CARD`        | Tarjeta de servicios públicos (Irlanda).                                      |
| `TYPE_RESIDENCE_PERMIT`            | Permiso de residencia.                                                        |
| `TYPE_RESIDENT_ID`                 | Identificación de residente.                                                  |
| `TYPE_TEMPORARY_RESIDENCE_PERMIT`  | Permiso de residencia temporal.                                               |
| `TYPE_VOTER_ID`                    | Identificación de votante.                                                    |
| `TYPE_WORK_PERMIT`                 | Permiso de trabajo.                                                           |
| `TYPE_iKAD`                        | Tarjeta para trabajadores extranjeros en Malasia.                             |
| `TYPE_MILITARY_ID`                 | Identificación militar.                                                       |
| `TYPE_MyKAS`                       | Tarjeta para residentes temporales en Malasia.                                |
| `TYPE_SOCIAL_SECURITY_CARD`        | Tarjeta de seguridad social.                                                  |
| `TYPE_HEALTH_INSURANCE_CARD`       | Tarjeta de seguro médico.                                                     |
| `TYPE_PASSPORT`                    | Pasaporte estándar.                                                           |
| `TYPE_S_PASS`                      | Permiso de trabajo para trabajadores calificados (Singapur).                  |
| `TYPE_ADDRESS_CARD`                | Tarjeta que certifica dirección de residencia.                                |
| `TYPE_ALIEN_ID`                    | Identificación para extranjeros.                                              |
| `TYPE_ALIEN_PASSPORT`              | Pasaporte para personas sin ciudadanía.                                       |
| `TYPE_GREEN_CARD`                  | Tarjeta de residencia permanente (EE. UU.).                                   |
| `TYPE_MINORS_ID`                   | Identificación para menores de edad.                                          |
| `TYPE_POSTAL_ID`                   | Identificación emitida por el servicio postal.                                |
| `TYPE_PROFESSIONAL_DL`             | Licencia de conducir profesional.                                             |
| `TYPE_TAX_ID`                      | Identificación fiscal.                                                        |
| `TYPE_WEAPON_PERMIT`               | Permiso de portación de armas.                                                |
| `TYPE_VISA`                        | Visa de entrada o estancia en un país.                                        |
| `TYPE_BORDER_CROSSING_CARD`        | Tarjeta para cruces fronterizos (como la BCC en EE. UU.).                     |
| `TYPE_DRIVER_CARD`                 | Tarjeta de conductor (Europa).                                                |
| `TYPE_GLOBAL_ENTRY_CARD`           | Tarjeta de viajero confiable (EE. UU.).                                       |
| `TYPE_MyPolis`                     | Identificación policial en Malasia.                                           |
| `TYPE_NEXUS_CARD`                  | Tarjeta de cruce rápido EE. UU.-Canadá.                                       |
| `TYPE_PASSPORT_CARD`               | Tarjeta de pasaporte para viajes terrestres/marítimos (EE. UU.).              |
| `TYPE_PROOF_OF_AGE_CARD`           | Tarjeta que acredita la mayoría de edad.                                      |
| `TYPE_REFUGEE_ID`                  | Identificación para refugiados.                                               |
| `TYPE_TRIBAL_ID`                   | Identificación emitida por una tribu reconocida.                              |
| `TYPE_VETERAN_ID`                  | Identificación para veteranos militares.                                      |
| `TYPE_CITIZENSHIP_CERTIFICATE`     | Certificado de ciudadanía.                                                    |
| `TYPE_MY_NUMBER_CARD`              | Tarjeta de número personal (Japón).                                           |
| `TYPE_CONSULAR_PASSPORT`           | Pasaporte emitido por una oficina consular.                                   |
| `TYPE_MINORS_PASSPORT`             | Pasaporte para menores de edad.                                               |
| `TYPE_MINORS_PUBLIC_SERVICES_CARD` | Tarjeta de servicios públicos para menores.                                   |
