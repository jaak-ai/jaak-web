title: Glosario
description: Definiciones de los términos esenciales utilizados en la plataforma y servicios de JAAK.
category: recurso
order: 1
---

* **Plataforma JAAK**:
* **Tecnología JAAK**: Toda aquella tecnología, proceso, metodología o sistema creado por la empresa JAAK.
* **Servicios JAAK**: Los componentes tecnológicos desarrollados por JAAK, y que son necesarios para el funcionamiento de sus diversas soluciones, se ponen a disposición a través de API.
* **KYC**: Know Your Customer. Conoce a tu cliente es un proceso de identificación y verificación de la identidad de un cliente o persona.
* **Sesión KYC**: Es el proceso para llevar a cabo un KYC mediante las herramientas de JAAK.
* **Token de Sesión KYC / accessToken**: También llamado AccessToken, es el token JWT generado a partir de un Short Key que se utiliza para autorizar y autenticar las llamadas a los servicios JAAK dentro de una sesión KYC.
* **Short Key**: Clave temporal y única que se genera al crear una sesión KYC para identificar y acceder a ella.
* **Geolocalización**: Proceso de registro de la ubicación geográfica.
* **Verificación de documentos (Servicio JAAK)**: Proceso en el que se valida que los documentos proporcionados por un cliente sean legítimos y auténticos.
* **Extracción de datos en documentos (Servicio JAAK)**: El proceso de extracción de información de un documento contenida en fotos o escaneos, se conoce como OCR.
* **Lista negra (Servicio JAAK)**: Base de datos que contiene información sobre individuos o entidades, actualmente las siguientes listas son soportadas, INE, RENAPO, SAT69b, OFAC, Interpol.
* **Verificación de vida o Liveness (Servicio JAAK)**: Proceso que confirma si el rostro capturado dentro de un video pertenece a una persona real.
* **Comparación OTO (Servicio JAAK)**: Proceso que compara dos rostros y define si son la misma persona. Dentro de nuestro KYC, este proceso se encarga de comparar la imagen del documento de identidad con la imagen capturada del usuario.
* **API**: Interfaz de Programación de Aplicaciones, por sus siglas en inglés: Application Programming Interface, es un conjunto de reglas y herramientas que permiten que diferentes aplicaciones o sistemas se comuniquen entre sí.
* **API key**: Clave de autenticación utilizada para acceder a los servicios de JAAK.
* **Endpoint**: Se utiliza para comunicarse con la API; cada endpoint puede representar una función o un recurso único que ofrece la API.
* **Base64**: Es un método de codificación que convierte datos binarios en una representación de texto utilizando solo caracteres ASCII seguros, comúnmente usado para transferir imágenes, archivos o datos en JSON y XML.
* **OCR (Optical Character Recognition)**: Es una tecnología que permite extraer y convertir texto de imágenes o documentos escaneados en datos editables y buscables. Se usa en aplicaciones como digitalización de documentos e identificación automática.
* **REST (Representational State Transfer)**: Es un estilo de arquitectura para servicios web que utiliza operaciones HTTP estándar (GET, POST, PUT, DELETE) para la comunicación entre clientes y servidores, manteniendo una estructura sencilla, escalable y sin estado.
