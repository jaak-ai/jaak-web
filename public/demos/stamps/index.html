📚 Guía de Uso - Jaak Stamps

🎯 ¿Qué es este componente?

Jaak Stamps es un componente web para capturar y procesar documentos de identificación. Detecta automáticamente el frente y reverso del documento, los alinea correctamente y genera imágenes optimizadas para su procesamiento.

🚀 Cómo usar

1. Precargar el modelo (opcional):

Haz clic en "⚡ Precargar" para cargar el modelo de detección antes de iniciar.

2. Iniciar captura:

Haz clic en "🎥 Iniciar" para activar la cámara y comenzar la detección.

3. Posicionar el documento:

  • Coloca el documento dentro del área marcada
  • Asegúrate de que esté bien iluminado
  • Evita reflejos y sombras
  • El componente detectará automáticamente cuando esté bien alineado

4. Captura automática:

El sistema capturará automáticamente ambos lados del documento cuando detecte la alineación correcta.

⚙️ Configuración

Puedes ajustar el comportamiento del componente usando el panel de configuración. Los cambios se pueden aplicar mediante el botón correspondiente o usando parámetros URL.

  • Debug: Muestra información de diagnóstico en pantalla
  • Tolerancia de alineación: Qué tan preciso debe estar el documento (0-100)
  • Tamaño de máscara: Porcentaje del área de captura (50-100%)
  • Margen de recorte: Píxeles adicionales alrededor del documento
  • Tiempo de captura: Milisegundos que el documento debe mantenerse alineado antes de capturar (0-10000ms)
  • Clasificación: Activa la clasificación automática del tipo de documento
  • Cámara: Selecciona entre cámara frontal o trasera
  • Versión del componente: Permite cambiar entre diferentes versiones del componente:
    • Local: Usa la versión de desarrollo local
    • Versiones específicas: Selecciona una versión específica publicada en npm
    • Buscar versión: Usa el campo de búsqueda para encontrar cualquier versión

🔗 Parámetros por URL

Puedes configurar el componente mediante parámetros en la URL:

?debug=false&alignmentTolerance=20&maskSize=85&captureDelay=2000&version=2.0.0-dev.38

📡 Registro de Eventos

Debajo del componente encontrarás el registro de eventos, que muestra en tiempo real todos los eventos emitidos:

  • captureCompleted: Se emite cuando se completa una captura
  • isReady: Se emite cuando el componente está listo para usar
  • traceIdGenerated: Se emite con el ID de rastreo generado o proporcionado

Puedes limpiar el historial de eventos con el botón "🗑️ Limpiar". El registro mantiene hasta 100 eventos, eliminando automáticamente los más antiguos.

📸 Imágenes Capturadas

Una vez completada la captura, aparecerá automáticamente debajo del registro de eventos un panel con todas las imágenes procesadas:

  • Imágenes completas del frame (frente y trasera)
  • Documentos recortados y alineados (frente y trasera)
  • Información de la captura (fecha, hora, total de imágenes)
  • Opciones para descargar cada imagen individualmente
  • Opción para ver las imágenes en tamaño completo
  • Descarga de todos los datos en formato JSON

El panel de imágenes se oculta automáticamente cuando se reinicia la captura.
