📚 Guía de Usuario - Jaak Visage

🎯 ¿Qué es este componente?

Jaak Visage es un componente web para detección facial y captura de video. Utiliza MediaPipe para detectar rostros, los rastrea en tiempo real, y automáticamente captura videos de 4 segundos cuando el rostro está posicionado correctamente.

🚀 Cómo usar

1. Precargar modelos (opcional):

Haz clic en "⚡ Preload" para cargar los modelos de detección antes de comenzar.

2. Iniciar detección:

Haz clic en "🎥 Start" para activar la cámara e iniciar la detección facial.

3. Posicionamiento del rostro:

  • Posiciona tu rostro dentro de la vista de la cámara
  • Asegúrate de tener buena iluminación
  • El componente detectará y rastreará rostros automáticamente
  • La grabación iniciará automáticamente cuando se detecte un rostro

4. Captura de video:

En modo automático, el sistema capturará automáticamente el video cuando se detecten rostros. En modo manual, usa el botón "📸 Capture Now" para controlar cuándo iniciar la grabación.

5. Mostrar instrucciones de ayuda:

Haz clic en "📚 Show Help" para mostrar las instrucciones de ayuda en cualquier momento.

⚙️ Configuración

Puedes ajustar el comportamiento del componente usando el panel de configuración. Los cambios se pueden aplicar usando el botón o mediante parámetros de URL.

  • Depuración: Muestra información de diagnóstico en pantalla (rotación facial: Roll, Pitch, Yaw, porcentaje de área)
  • Mostrar ayuda al iniciar: Muestra automáticamente las instrucciones de ayuda al iniciar la detección
  • Cámara: Selecciona entre cámara frontal, trasera, o selección automática
  • Modo de Detección: Configura el rendimiento de detección facial:
    • Completo: Máxima precisión con modelo personalizado (ideal para equipos potentes)
    • Ligero: Optimizado para dispositivos móviles con recursos limitados
    • Automático: Detecta automáticamente las capacidades del dispositivo y selecciona el mejor modo
  • Modo de Captura: Controla cuándo se inicia la grabación:
    • Automático: La grabación se inicia automáticamente cuando se detecta un rostro en posición óptima
    • Manual: El usuario controla cuándo iniciar la grabación usando el botón "📸 Capture Now"
  • Idioma: Selecciona el idioma de los mensajes e instrucciones:
    • Español: Todos los textos se muestran en español (predeterminado)
    • English: All texts are displayed in English
  • Versión del Componente: Cambia entre diferentes versiones:
    • Local: Usa la versión de desarrollo local
    • Versiones específicas: Selecciona una versión publicada en npm
    • Búsqueda: Usa el campo de búsqueda para encontrar cualquier versión

🔗 Parámetros de URL

Configura el componente mediante parámetros de URL:

?debug=false&showHelpOnStart=true&camera=back&detectionMode=auto&captureMode=manual&language=en&version=1.0.0

📡 Registro de Eventos

Debajo del componente encontrarás el registro de eventos, que muestra en tiempo real todos los eventos emitidos:

  • videoCaptured: Se emite cuando se completa una captura de video
  • statusUpdated: Se emite cuando el estado del componente cambia
  • traceIdGenerated: Se emite con el ID de rastreo generado o proporcionado

Puedes limpiar el historial de eventos con el botón "🗑️ Limpiar". El registro mantiene hasta 100 eventos, eliminando automáticamente los más antiguos.

💾 Resultados

Después de completar la captura, verás el video grabado con:

  • Video de 4 segundos en formato WebM (códec VP9)
  • Resolución 4K (4096x2160) cuando esté disponible
  • Velocidad de fotogramas de 30 FPS
  • Controles de reproducción
  • Metadatos de grabación
